# Rules
- Write **unit tests** using `XCTest`.
- Use **XCUITest** for UI testing.
- Cover **common user flows** with tests.
- Perform **performance and stress testing**.
- Simulate **error handling** scenarios.
- Debugging should follow the **Debugger Mode process**:
  1. Identify **5-7 possible sources** of the issue.
  2. Narrow down to **1-2 likely causes**.
  3. Add **logs to validate assumptions**.
  4. Analyze **console and network logs**.
  5. Suggest **additional logs** if needed.
  6. Propose a **comprehensive fix**.
  7. Once fixed, **remove unnecessary logs**.
